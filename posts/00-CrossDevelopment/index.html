What is Cross Development?

Cross DevelopmentлһҖ лӮҙк°Җ м§ҖкёҲ к°ңл°ңн•ҳкі  мһҲлҠ” нҷҳкІҪкіј мӢӨм ң к°ңл°ңлҗң м–ҙн”ҢлҰ¬мјҖмқҙм…ҳмқҙ лҸҷмһ‘н•ҳлҠ” нҷҳкІҪмқҙ лӢӨлҘҙкІҢ м–ҙн”ҢлҰ¬мјҖмқҙм…ҳмқ„ к°ңл°ңн•ҳлҠ” кІғмқ„ мқҳлҜён•ңлӢӨ.

мҳҲлҘј л“Өл©ҙ Windows 10 нҷҳкІҪм—җм„ң Visual StudioлЎң Windows 10м—җм„ң лҸҷмһ‘н•ҳлҠ” м–ҙн”ҢлҰ¬мјҖмқҙм…ҳмқ„ л§Ңл“Ө мҲҳлҸ„ мһҲм§Җл§Ң, н”„л Ҳмһ„мӣҢнҒ¬мқҳ лҸ„мӣҖмқ„ л°ӣм•„ OS л ҲлІЁм—җм„  iOS, Androidл“ұм—җм„ң лҸҷмһ‘н•ҳлҠ” м–ҙн”ҢлҰ¬мјҖмқҙм…ҳлҸ„ л§Ңл“Ө мҲҳ мһҲлӢӨ. Xamarin, UWP, Flutterл“ұмқҙ к·ё н”„л Ҳмһ„мӣҢнҒ¬мқҳ мҳҲмқҙлӢӨ.

нҷҳкІҪмқҙ лӢӨлҘҙлӢӨлҠ” кІғмқҖ м–ҙл–ӨкІғмқ„ мқҳлҜён• к№Ң?

нҳ№мһҗлҠ” вҖңWindows кұҙ Macмқҙкұҙ Androidкұҙ мҪ”л“ңм§Өл•ҢлҠ” лӢӨ лҳ‘к°ҷмқҖлҚ°, м–ҙн”ҢлҰ¬мјҖмқҙм…ҳ л ҲлІЁм—җм„ң мӣҗн•ҳлҠ” кё°лҠҘ лӢӨ лҳ‘к°ҷмқҖлҚ° лӯҗк°Җ лӢӨлҘёкұ°м•ј?вҖқ н•ҳкі  мғқк°Ғн•  мҲҳ мһҲлӢӨ.

н•ҳм§Җл§Ң лӮҙл¶ҖлЎң л“Өм–ҙк°Җл©ҙ к·ё м–ҙн”ҢлҰ¬мјҖмқҙм…ҳмқҙ лҸҷмһ‘н•ҳкё°мң„н•ҙ н•„мҡ”лЎңн•ҳлҠ” лқјмқҙлёҢлҹ¬лҰ¬л“Өмқҙ мһҲкі , к·ё лқјмқҙлёҢлҹ¬лҰ¬л“Өмқҙ лҸҷмһ‘н•ҳкё° мң„н•ң л””л°”мқҙмҠӨл“Өкіј к·ё кІғл“Өмқ„ мҡҙмҡ©н•ҳлҠ” м»Өл„җ л°Ҹ мҡҙмҳҒмІҙм ң к·ёлҰ¬кі  м•„нӮӨн…Қміҗл“Өмқҙ мӨҖл№„к°Җ лҗҳм–ҙм•ј л№„лЎңм„ң м–ҙн”ҢлҰ¬мјҖмқҙм…ҳмқҙ лҸҷмһ‘ н•  мҲҳ мһҲлҠ” кІғмқҙлӢӨ.

мқҙлҹ¬н•ң мӢңмҠӨн…ң л ҲлІЁмқҳ мһ‘м—…л“Өмқ„ 1м°Ём ҒмңјлЎң мІҳлҰ¬н•ҙмЈјлҠ” кІғмқ„ мҡ°лҰ¬лҠ” н”„л Ҳмһ„мӣҢнҒ¬лӢЁлқјкі  л¶ҖлҘҙкё°лҸ„ н•ңлӢӨ.

  • JAVAмқҳ JavaRuntimeEnvrionment
  • Windowsмқҳ .NETFramework
  • MACмқҳ Rosetta 2

к·ёлҰ¬кі  Visual Studio, Android Studio к°ҷмқҖ нҶөн•©к°ңл°ңнҷҳкІҪ(IDE)м—җм„ң к°ңл°ңмһҗл“Өмқҙ мүҪкІҢ к°ңл°ңн•  мҲҳ мһҲкІҢ м»ҙнҢҢмқјлҹ¬, л§Ғм»Ө, л””лІ„кұ° л“ұмқ„н•ҳм—¬ н”„л Ҳмһ„мӣҢнҒ¬мҷҖ м—°кІ°мӢңмјңмЈјлҠ” кІғмқҙлӢӨ.

нҳ№мһҗлҠ” вҖңк·ёлҹј м ҖлҠ” м»ҙнҢҢмқјмқҙ н•„мҡ”м—ҶлҠ” нҢҢмқҙмҚ¬, JavaScript, Ruby, SQL кіј к°ҷмқҖ мқён„°н”„лҰ¬н„° м–ём–ҙлҘј мқҙмҡ©н•ҳм—¬ к°ңл°ңн•  кұҙлҚ°, Cross Developmentн•  н•„мҡ”м—ҶлҠ”кұ°л„Өмҡ”?вҖқ н•ҳкі  л§җн•  мҲҳ мһҲлӢӨ.

мһ‘м„ұн•ң мқён„°н”„лҰ¬н„° м–ём–ҙм—җм„ң н•„мҡ”лЎң н•ҳлҠ” лқјмқҙлёҢлҹ¬лҰ¬к°Җ лҸҷмһ‘н•ҳкі мһҗ н•ҳлҠ” нҷҳкІҪмқ„ м§Җмӣҗн•ҳм§Җ м•ҠлҠ”лӢӨл©ҙ мқҙ м–ём–ҙлЎң мһ‘м„ұн•ң м–ҙн”ҢлҰ¬мјҖмқҙм…ҳмқҖ л¬ҙмҡ©м§Җл¬јмқҙлӢӨ.

мқҙл Үл“Ҝ мҡ°лҰ¬к°Җ Thoroughн•ҳкІҢ нҷҳкІҪл“Өмқ„ кі л Өн•ҙм•ј м–ҙн”ҢлҰ¬мјҖмқҙм…ҳмқҳ лҸҷмһ‘кіј м„ұлҠҘмқ„ н…ҢмҠӨнҠён•ҳкі  ліҙмһҘн•  мҲҳ мһҲлӢӨ.

Cross Compiling for Embedded System

Ref: Linux Foundation 2020 Conference

мһ„лІ л””л“ңлҠ” мқҙлҹ¬н•ң Cross Development кё°лІ•мқ„ мӮ¬мҡ©н•ҙм•ј нҡЁмңЁм Ғмқё л¶„м•ј мӨ‘ н•ҳлӮҳлӢӨ.

м§ҖкёҲмқҙм•ј мһ„лІ л””л“ң кё°кё°к°Җ нӣҢлҘӯн•ң м„ұлҠҘмқ„ лӮҙм§Җл§Ң, лӘҮл…„м „л§Ң н•ҳлҚ”лқјлҸ„ н•ҙлӢ№кё°кё°м—җм„ң м§Ғм ‘ к°ңл°ңмқ„ н•ңлӢӨкұ°лӮҳ Compileмқ„ н•ңлӢӨкұ°лӮҳ н•ҳлҠ” кІғмқҖ кҝҲлҸ„ кҫёкё° нһҳл“Өм—ҲлӢӨ.

к·ёлһҳм„ң л°ңм „н•ң Cross л№Ңл“ң мӢңмҠӨн…ңкҙҖл Ён•ң Open Source Projectsл“ӨмқҖ OpenEmbedded, Buildroot, Yocto Project, Bitbake л“ұмқҙ мһҲлӢӨ.
