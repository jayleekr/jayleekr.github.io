## POSIX에 대하여

- 일반
    - Portable Operating System Interface (이식 가능한 운영 체제 인터페이스)
    - 1988년부터~
    - 서로 다른 운영 체제가 서로 다른 API를 사용하여 개발에 어려움을 겪는 문제를 해결하기 위해 최소한의 영역이라도 통일하자는 아이디어에서 시작
    - 사양: [http://get.posixcertified.ieee.org/](http://get.posixcertified.ieee.org/)
    - 버전
        - POSIX.1 (IEEE 1003.1-1988): 핵심 서비스
        - POSIX.2 (IEEE 1003.2-1992): 셸 및 유틸리티
        - POSIX.1b (IEEE 1003.1b-1993): 실시간 관련
        - POSIX.1c (IEEE 1003.1c-1995): 스레드 관련
    - POSIX.1 - 2017까지 출시됨
        - 프로세스 생성 및 제어, 시그널, 파일 및 디렉토리 작업, 파이프, C 라이브러리, I/O 포트 인터페이스 및 제어, 프로세스 트리거

- POSIX PSE51 호환 API
    - IEEE 1003.13에 정의됨
    - POSIX 실시간 및 임베디드 애플리케이션 지원 (AEP)
    - PSE51-최소 실시간 시스템 프로필: 소형 임베디드 시스템, MMU 없음, 디스크 없음, 터미널 없음
