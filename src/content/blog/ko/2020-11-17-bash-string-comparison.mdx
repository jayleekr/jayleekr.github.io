## Bash "if"를 사용한 문자열 비교

"=="와 "!="는 문자열 비교의 경우에만 사용할 수 있습니다.

``` sh
if [ "$STRING" == "abc" ];then
    echo "STRING은 abc입니다!"
fi 

또는

if [ "$STRING" = "abc" ];then
    echo "STRING은 abc입니다!"
fi 
```
