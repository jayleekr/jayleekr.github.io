## Get Ip Address

만약 ethernet interface가 eth0 인 경우

``` sh
$ IPADDR=$(ifconfig eth0|grep inet|head -1|sed 's/\:/ /'|awk '{print $2}')
$ echo $IPADDR
172.17.0.3
```
