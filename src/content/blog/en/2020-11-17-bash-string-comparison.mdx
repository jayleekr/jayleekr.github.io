## Bash string comparison using "if"

"==" and "!=" only can be used in case of string comparison.

``` sh
if [ "$STRING" == "abc" ];then
    echo "STRING is abc!"
fi 

or

if [ "$STRING" = "abc" ];then
    echo "STRING is abc!"
fi 
```
