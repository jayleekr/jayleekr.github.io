# Project Overview

## Purpose
Personal developer blog (jayleekr.github.io) focusing on software engineering, embedded systems, and technology insights. Built with Astro for optimal performance and modern developer experience.

## Tech Stack
- **Framework**: Astro 5.x (Static Site Generator)
- **Language**: TypeScript + JavaScript
- **Styling**: Tailwind CSS with Typography plugin
- **Content**: MDX (Markdown + Components)
- **Testing**: Vitest (unit), Playwright (E2E), Lighthouse (performance)
- **Hosting**: GitHub Pages with automated deployment
- **Analytics**: Google Analytics 4
- **Comments**: Giscus (GitHub Discussions)
- **Package Manager**: Bun (with npm fallback)
- **Build Tool**: Vite (via Astro)

## Key Features
- Multi-language support (Korean/English) via astro-i18next
- Content collections with categories and tags
- SEO optimization (JSON-LD, Open Graph, sitemap)
- Accessibility: WCAG 2.1 AA compliant
- Performance: 96/100 Lighthouse score
- Dark/light theme toggle with system preference
- Site-wide search functionality
- RSS feed generation
- Progressive Web App (PWA) capabilities
- Reading time estimation
- Table of contents auto-generation
- Responsive mobile-first design

## Architecture
- Static site generation (SSG)
- Component-based architecture
- Content collections for blog posts
- File-based routing in src/pages/
- Organized by categories: TechSavvy, DeepThinking, Collaboration
